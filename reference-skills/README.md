# Reference Skills

## Don't peek until Part 4!

These files contain reference implementations of the email and LinkedIn ghost writer skills. They're here so you can compare your work against a polished version — but only AFTER you've built your own.

**Why wait?**

The whole point of this session is to experience Claude building a skill from writing samples. If you read the reference files first, you'll anchor on this structure instead of discovering what Claude creates organically. The learning is in the building, not the reading.

**When to use these files:**

- In **Part 4** of the handbook, you'll compare your skill files against these references
- After the session, use them as templates for building new skills
- If you get stuck during the session, ask your instructor before peeking here

**Files:**

| File | What It Is |
|------|-----------|
| `email-ghostwriter.md` | Reference skill for generating emails in a specific voice |
| `linkedin-ghostwriter.md` | Reference skill for generating LinkedIn posts in a specific voice |

These are skill files — markdown files designed to live in `.claude/commands/` and be invoked with a slash command like `/email-ghostwriter` or `/linkedin-ghostwriter`.
